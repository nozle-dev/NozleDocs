## Nozle Cloud (Recommended)

The fastest path to production is Nozle Cloud — the fully managed platform at [app.nozle.app](https://app.nozle.app). It includes:

- Margin intelligence and cost model analytics
- Real-time entitlement checks with low-latency caching
- Stripe embedded checkout
- Full SDK support (React, Node.js, Python)
- Automatic scaling, monitoring, and updates

No infrastructure to manage. Sign up and start billing in minutes.

## Self-Hosting

If you need to self-host for compliance or data residency reasons, contact us at [app.nozle.app](https://app.nozle.app) for deployment guides and support.

Self-hosted deployments provide invoicing, subscriptions, and event aggregation. For margin intelligence, real-time entitlements, and the React SDK, connect your self-hosted instance to Nozle Cloud.
