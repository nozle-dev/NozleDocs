---
title: "WebSocket Token"
api: "GET /auth/centrifugo-token"
---

Get a WebSocket authentication token for real-time entitlement updates via Centrifugo.

**Auth:** secret key (`sk_`) only.

Publishable keys and customer sessions cannot mint realtime tokens. Browser customer-credit access is HTTP-only.

<ResponseField name="token" type="string" required>
  JWT token for authenticating WebSocket connections to Centrifugo
</ResponseField>
