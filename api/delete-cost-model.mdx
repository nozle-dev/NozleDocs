---
title: "Delete Cost Model"
api: "DELETE /cost-models/{id}"
---

Delete a cost model by ID.

**Auth:** `sk_` only

<Warning>
All cost model endpoints require a secret key (`sk_`). Deletion is constrained to that key's organization; an ID from another organization is not accessible. Publishable keys and customer sessions receive a 403 Forbidden response.
</Warning>

<ParamField path="id" type="string" required>
  Cost model ID
</ParamField>
