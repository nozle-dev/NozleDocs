---
title: "List Invoices"
api: "GET /invoices"
---

List all invoices for a customer. Returns invoices in reverse chronological order.

**Auth:** customer session with `billing:read`, or a secret key. Publishable keys are always rejected.

<ParamField query="customer_id" type="string" required>
  External customer ID. For a customer session, this must match the token subject.
</ParamField>

### Response

Returns an array of invoice objects including status, amounts, and payment details.
