---
title: Billing API
description: Full billing engine API for managing customers, plans, subscriptions, invoices, and more
---

The Billing API provides full CRUD access to every billing resource. Use it to programmatically manage your billing configuration — customers, plans, subscriptions, invoices, wallets, coupons, and more.

## Base URL

```
https://billing.nozle.app/api/v1
```

## Authentication

All Billing API requests require a Bearer token:

```
Authorization: Bearer YOUR_BILLING_API_KEY
```

Get your billing API key from the Nozle dashboard under **Settings > API Keys**.
