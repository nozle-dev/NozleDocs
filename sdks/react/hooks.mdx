## usePlans

Fetches the public plan catalog using the provider's `pk_`.

```tsx
const { plans, isLoading } = usePlans()
```

## useCheckoutSession

Invokes the merchant-provided `createCheckout` callback. The hook passes only `planCode` and `returnUrl`; it does not send a customer ID.

```tsx
const { fetchClientSecret, checkout, isLoading, error } = useCheckoutSession()

const clientSecret = await fetchClientSecret(
  'pro',
  'https://wrrk.ai/settings/billing',
)
```

The callback may return a Stripe client secret or URL, `type: "completed"`, or `type: "scheduled"`. If the callback is missing, the hook rejects with a configuration error.

Customer billing, invoices, cancellation, top-ups, subscriptions, entitlement checks, and credit reads are intentionally absent from the browser SDK. Implement authenticated merchant endpoints and pass their results into presentational components.
